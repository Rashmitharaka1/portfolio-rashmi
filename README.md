# 🚀 Getting Started

Follow these steps to run the project locally.

## 1️⃣ Clone the Repository

```bash
git clone https://github.com/Rashmitharaka1/portfolio-rashmi.git
```

---

## 2️⃣ Navigate to Project Folder

```bash
cd portfolio-rashmi
```

---

## 3️⃣ Install Dependencies

```bash
npm install
```

---

## 4️⃣ Run Development Server

```bash
npm run dev
```

---

## 5️⃣ Open in Browser

```bash
http://localhost:5173
```

---

# 📦 Build for Production

```bash
npm run build
```
